Table Tags üìù

Tag Name Description
<table> Table Defines the table structure.
<tr> Table Row Defines a row within a table.
<td> Table Data Defines a standard data cell within a row.
<th> Table Header Defines a header cell within a row.
<thead> Table Head Groups the header content in a table.
<tbody> Table Body Groups the body content in a table.
<tfoot> Table Footer Groups the footer content in a table.
<caption> Table Caption Adds a caption to a table.
<colgroup> Column Group Defines a group of columns within a table for styling purposes.
<col> Column Specifies column properties for styling within a <colgroup>.
